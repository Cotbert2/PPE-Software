Proyecto

El proyecto presente plantea el brindar un curso introductorio al desarrollo de software, con una ruta más planeada hacia el desarrollo de sitios y aplicaciones web, con un temario centralizado en 3 partes (HTML5, CSS3 y Programación)los cuales permitirán al estudiante el fácil aprendizaje de lenguajes Frontend con tutoriales realizados por estudiantes del club de participación estudiantil.

  • 📝 Proyecto



    • Malla:

    • HTML 🔽

      • ▶️ ¿Qué es html y para que sirve?
      • ▶️ Instalación de un editor de textos(Visual Studio Code)
      • ▶️ Mi primera página web: Estructura básica en HTML 5
      • ▶️ Ubicar imagenes en HTML 5
      • ▶️ Headings en HTML5

    • CSS 🔽

      • ▶️ ¿Qué es CSS y para qué sirve?
      • ▶️ Cambiar el color de fondo y de una fuente
      • ▶️ Cambiar el tipo de fuente con Google Fonts
      • ▶️ Alineación y tamaño de un texto
      • ▶️ PROYECTO: Creación de una barra de navegación

    • Programación 🔽

      • ▶️ Desarrollo de una Calculadora en Raptor
      • 📜 Frameworks de Javascript
      • 📜 Lenguajes de programación más usados
      • 📝 Exámen Final
